# Features
- Bootstrap 4.1.3
- Bootstrap 4 Validation
- Verified by W3C
- jQuery 1.12.4
- 4 Pages included (Login, Register, Forgot Password, Reset Password)
- Clean design
- Show/hide password
- Easy to use
- Cross-browser compatibility (Chrome, Firefox, Opera, IE11, Safari)
- Fully responsive
- and more ...
